# FightSpade

A mod which allows you to fight a cut boss fight against Spade.

To fight Spade, head to Shang Tu Dojo - he replaces the Askal fight.
Mod uses music and sounds for Freedom Planet 1 in cases where FP2 lacked these.

This mod is compatible with Music Replacer - you can replace the track played during the fight by replacing "M_Boss_Spade".

Install:
Use FP2 Mod Installer

Manual Install:

    Install BepInEx 5 (Download) - extract the downloaded file into main game directory.
    Install the mod - extract the zipfile contents into the main game dir - (if asked, agree to merge the BepInEx folders).
